## HIPOSTESIS
Creemos que los usuarios que realizan trámites o actividades en entidades públicas o privadas necesitan conocer si existe una cola antes de acudir al establecimiento y utilizarán una aplicación móvil que muestre el estado de las filas en tiempo real, lo cual se evidenciará mediante consultas frecuentes a la aplicación para planificar sus visitas y por tanto sus actividades diarias.
